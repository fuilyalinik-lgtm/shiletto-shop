# Мій Дропшифтинг Сайт

Це динамічний сайт для дропшиппінгу з системою аутентифікації та адмін-панеллю.

## Локальний запуск

1. Встановіть Python (якщо не встановлений) з https://www.python.org (позначте "Add Python to PATH").
2. Запустіть відповідний скрипт:
   - **Windows**: Двічі клацніть `start_server.bat`
   - **Linux/Mac**: Виконайте `./start_server.sh` (зробіть виконуваним: `chmod +x start_server.sh`)
   
   Скрипт автоматично створить віртуальне оточення, встановить залежності та запустить сервер.
3. Відкрийте http://localhost:8080

Альтернативно:
- Перейдіть у папку `backend`
- Активуйте віртуальне оточення: `.venv\Scripts\activate` (Windows) або `source .venv/bin/activate` (Linux/Mac)
- Встановіть залежності: `pip install -r requirements.txt`
- Запустіть сервер: `python app.py` (або `python3`)

## Структура

- `backend/` - Flask додаток з API
- `frontend/` - HTML/CSS/JS файли
- `start_server.bat` - скрипт запуску для Windows
- `start_server.sh` - скрипт запуску для Linux/Mac

## Функції

- Реєстрація та вхід за email
- Перегляд категорій та товарів
- Адмін-панель для управління (доступ тільки адміну)
- Інтерактивна карта на сторінці контактів

## Розгортання

Завантажте на хостинг з підтримкою Python/Flask.
